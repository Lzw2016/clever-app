返回目录 表名：task_job
说明：定时任务
表字段：
序号 名称 数据类型 主键 不能为空 是否索引 默认值 说明
1 id bigint(20) Y(自增) Y Y(唯一) 主键id
2 namespace varchar(63) N Y 命名空间
3 name varchar(127) N Y Y 任务名称
4 type tinyint(4) N Y 任务类型，1：http调用，2：java调用，3：js脚本，4：shell脚本
5 max_reentry tinyint(4) N Y 0 最大重入执行数量(对于单个节点当前任务未执行完成就触发了下一次执行导致任务重入执行)，小于等于0：表示禁止重入执行
6 allow_concurrent tinyint(4) N Y 1 是否允许多节点并发执行，使用悲观锁实现，不建议禁止，0：禁止，1：允许
7 max_retry_count int(11) N Y 0 执行失败时的最大重试次数
8 route_strategy tinyint(4) N Y 0 路由策略，0：不启用，1：指定节点优先，2：固定节点白名单，3：固定节点黑名单
9 first_scheduler varchar(2047) N N 路由策略，1-指定节点优先，调度器名称集合
10 whitelist_scheduler varchar(2047) N N 路由策略，2-固定节点白名单，调度器名称集合
11 blacklist_scheduler varchar(2047) N N 路由策略，3-固定节点黑名单，调度器名称集合
12 load_balance tinyint(4) N Y 1 负载均衡策略，1：抢占，2：随机，3：轮询，4：一致性HASH
13 is_update_data tinyint(4) N Y 1 是否更新任务数据，0：不更新，1：更新
14 job_data text N N 任务数据(json格式)
15 run_count bigint(20) N Y 0 运行次数
16 disable tinyint(4) N Y 0 是否禁用：0-启用，1-禁用
17 description varchar(511) N N 描述
18 create_at datetime(3) N Y Y CURRENT_TIMESTAMP(3) 创建时间
19 update_at datetime(3) N N Y 更新时间
表索引：
序号 索引名 索引字段 唯一索引
1 idx_task_job_create_at create_at N
2 idx_task_job_name name N
3 idx_task_job_update_at update_at N
4 primary id Y
